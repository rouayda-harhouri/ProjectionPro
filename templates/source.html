Description du projet

Ce projet est une application web qui permet de convertir des coordonnées géographiques entre différents systèmes de projection (Mercator, Robinson, Lambert, etc.) et d'afficher les résultats sur des cartes interactives. Les utilisateurs peuvent sélectionner un système de projection d'entrée et un système de projection de sortie, et voir les résultats visualisés sur deux cartes distinctes.

Les technologies mentionnées ci-dessus sont combinées pour fournir une expérience utilisateur fluide et interactive. Bootstrap est utilisé pour un design élégant et responsive, et Leaflet permet d'intégrer des cartes interactives.

Technologies Utilisées

  • Python : Langage de programmation utilisé pour la logique backend.
  • Flask : Framework léger en Python pour le développement d'applications web.
  • PyProj : Une bibliothèque Python puissante utilisée pour effectuer des transformations entre différents systèmes de coordonnées géographiques et projections. Elle garantit des conversions précises en utilisant des paramètres de projection conformes aux normes.
  • Folium : Une bibliothèque Python pour créer des cartes interactives. Elle génère des cartes HTML dynamiques en intégrant des outils comme Leaflet, permettant d'afficher facilement les points de conversion et les systèmes de projection.
  • Jinja : Moteur de templating intégré à Flask, pour générer dynamiquement le contenu HTML.
  • Gestion d'authentification : Utilisation de sessions Flask pour gérer l'état de connexion des utilisateurs.
  • HTML & CSS : Pour structurer et styliser les pages web.
  • Bootstrap : Framework CSS pour la mise en page responsive et le design esthétique des pages.

Schéma de la Structure du Projet

Structure du projet

Le projet est organisé comme suit :

  • app.py : Fichier principal contenant le code backend en Flask.
  • requirements.txt : Liste des librairies python externes utilisées dans le projet.
  • templates/ : Dossier contenant les fichiers HTML (gérés avec Jinja).
  • static/ : Dossier pour les fichiers statiques, tels que CSS, JavaScript et images.
